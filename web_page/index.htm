Welcome to a University of Pennsylvania custom modbus gateway! This was designed as a translator from different dialects of Modbus to a more traditional form that could be easily understood by current metering software.


  • General Setup contains settings pertaining to the gateway itself.

  • Meter Setup contains setting pertaining to meters assigned to this gateway. Please note that there may be more meters connected to this meter. If the gateway receives a modbus request for an id that is not listed, then it will default to sending a request over rs485 wire.

  • Live Data is where snapshots of meters listed under Meter Setup can be found.

  • Download Data contains csv files of data collected at 5 minute intervals of the meters in Meter Setup.

  • Graph Data displays a graph of same data located in Download Data.
